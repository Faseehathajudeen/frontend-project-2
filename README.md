# frontend-project-2
Travel Explorer 🌍

Travel Explorer is a responsive travel website developed using HTML and CSS. The website is designed to provide users with an attractive and user-friendly platform to explore popular travel destinations.

Features

- Responsive Web Design
- Navigation Bar
- Hero Section
- Destination Cards
- Mobile-Friendly Layout
- Clean and Modern Interface

Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

Purpose

This project was created to practice responsive web design concepts and CSS layout techniques as part of a Frontend Development training program.
